# Oráculo Holístico 

Este é o repositório do site **Oráculo Holístico**, projeto idealizado por Talita Policarpo.  
O projeto reúne diferentes leituras de Tarot e práticas espirituais, com um design acolhedor e responsivo.

## Funcionalidades
- Catálogo de leituras com cartas ilustradas
- Seção "Sobre mim" com texto justificado
- Paleta de cores personalizada (#EEA9B8 e #a79371)
- Layout responsivo para desktop e mobile
- Botão de contato via WhatsApp


## Como visualizar
 Clone este repositório:
   ```bash
   git clone https://github.com/seuusuario/oraculo-holistico.git
```
